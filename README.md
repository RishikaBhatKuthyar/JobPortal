# Job Portal Overview

**React Job Portal** is a web application designed to bridge the gap between job seekers and employers. Leveraging React for the frontend, Material UI for design, Axios for API requests, and a Node.js backend, this portal offers an intuitive interface for users to explore job listings and company profiles efficiently.

---

## 🔑 Key Features

- **User Authentication**: Secure login with session management.
- **Dynamic Job Listings**: View job roles with details like required skills and salary.
- **Company Showcase**: Gallery view of participating companies.
- **Contact Form**: Quickly reach out to the portal admins.
- **Responsive Design**: Built using Material UI for seamless mobile and desktop use.

---

## 🛠️ Technical Stack

- **Frontend**: React, Material UI, Axios  
- **Backend**: Node.js, Express, MongoDB, Mongoose, bcrypt (for hashing), multer (for image uploads)  
- **Others**: CORS, dotenv

---

## ⚙️ Installation

> Ensure Node.js, npm/yarn, and MongoDB are installed.

### Backend Setup
```bash
nodemon run start
```

### Frontend Setup
```bash
npm install
npm start
```

The application will now be accessible at: [http://localhost:3000](http://localhost:3000)

---

## 🚀 Usage

After installation, log in with the credentials configured in the backend.

Navigate the app via the navbar:

- **Home**: Entry point
- **About**: Learn about the mission and vision
- **Job Listings**: Browse available jobs
- **Company Showcase**: View participating companies
- **Contact**: Reach out to administrators
